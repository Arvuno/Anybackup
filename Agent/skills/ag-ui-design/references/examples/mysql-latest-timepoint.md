## 显示目标
帮助用户确认“最新 MySQL 备份时间点”的查询进度和可见结论。

## 内容约束
展示目标对象为“目标 MySQL 实例”，展示查询范围为“最近一次可用备份时间点”。只展示脱敏时间点、可用性状态和下一步。

## 布局约束
根布局使用 `stack`。思考摘要使用 `heading` + `paragraph`；工具调用使用 `heading` + `kv-list` + `data-table`；结果使用 `callout` + `kv-list` + `action-row`。

## 思考链展示
展示“正在确认目标和最新时间点范围”，下一步为执行恢复点查询。不展示内部推理链。

## 工具调用展示
业务能力名称显示为“恢复点查询”。字段包含状态、调用目的、检查范围和脱敏输入摘要。

## 结果返回展示
标题为“已找到最新可用恢复点”。摘要展示最近时间点、目标对象、可用性、建议动作。提供“确认使用该时间点”和“查看候选时间点”动作。

## 禁止展示
不得展示内部索引、内部能力名、对象类标识、原始参数、原始返回、连接串、日志或堆栈。
